=== SysView Extension — Installation dans Brave ===

1. Ouvrir brave://extensions
2. Activer "Mode développeur" (coin haut droite)
3. Cliquer "Charger l'extension non empaquetée"
4. Sélectionner le dossier SysViewExtension
5. L'extension est active — icône SysView dans la barre

L'extension envoie automatiquement les données de lecture
vers SysViewBridge toutes les 1000ms (1s).
Position, durée, pause/lecture, miniature — en temps réel.
